SOBRE O PROJETO

Este projeto tem como objetivo principal o desenvolvimento de um site que melhore o processo de aprendizagem dos estudantes de cálculo 2. O foco do projeto é criar um site completo que apoiará a docente no ensino da disciplina por meio de conteúdos teóricos e de aulas remotas que possibilitem o aprendizado de forma atrativa e eficiente. Este site contará ainda com contribuições externas de outros professores e será divulgado para outras instituições de ensino com o intuito de facilitar o aprendizado de um público cada vez maior.
